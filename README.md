# disease-emergence
This repository contains all the necessary code and model development to conduct the analyses presented in Hirst et al, 2026. (enter citation)

Code for analysing DRC data and generating manuscript figures are found in `disease-emergence/code`.
Demographic data on age structure in the DRC is saved to `disease-emergence/data`.
Estimated loss of herd immunity through time and raw manuscript and supplemental figures are saved to `disease-emergence/results`
Final figure versions and mansucript are saved to `disease-emergence/products`

Project workflow: 
  `disease-emergece/code/data analysis/` -> `disease-emergence/code/project`
  See README in each folder for instructions on running the scripts. 
  

Note: This project has saved all package versions used in the original analysis with `renv`.
