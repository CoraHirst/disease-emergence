`disease-emergence/results` contains 3 subfolders:
    1) `data analysis`,
    2) `paper figs`, and
    3) `supplement`
    
`data analysis` contains the output of `disease-emergence/code/data analysis/congo/matrix_model_projections_2.qmd`
Population number by age through time: `population_matrix.csv` 
population immunity through time: `proportion_imnume.csv`

`paper figures` contains the raw output of manuscript figures generated in `disease-emergence/code/project/paper figures/paper_figures.qmd`

`supplement` contains the raw output of supplemental figures generated in `disease-emergence/code/project/paper figures/paper_figures.qmd`
